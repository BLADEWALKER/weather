## 项目依赖以及规范说明

### 依赖的第三方库
* 整体框架spring web mvc 4.3.2.RELEASE，[参考文档](http://spring.io/guides/gs/rest-service/)
* HTML Parser Jsoup-1.10.1 [参考文档](https://jsoup.org/)

### 编码规范
* 所有变量采用驼峰命名方式，常量全部采用大写
* 方法定义时大括号须换行，如：

    void fun(){
      //your code here
    }
* 操作数与运算符之间须留有空格，如 int a = b + c;

### 开发规范
* 提交时应添加描述信息
* 尽量多次分步提交，不要很过长时间再提交
* 提交之前删掉多余的注释和调试输出语句
